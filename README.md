# library

---

## API Endpoint Summary

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/libraries/books` | Get books by library and aisle name (with pagination) |
| POST | `/api/libraries` | Create a new library |
| PUT | `/api/libraries/{id}` | Update library information |
| GET | `/api/libraries/complete` | Get all libraries with aisles and books |
